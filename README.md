# Render Zwitch Service

Render start command:

```bash
npm install
npm start
```

Webhook URL after deploy:

```txt
https://YOUR_RENDER_APP.onrender.com/zwitch/payout-webhook
```

Flutter/Admin base URL:

```txt
https://YOUR_RENDER_APP.onrender.com
```

All admin APIs require:

```txt
Authorization: Bearer <Firebase ID token>
```

Environment variables are in `.env.example`.
